## What's Included

- Web search and scraping
- AI analysis and synthesis
- Structured data extraction
- Multi-source aggregation

## Basic Research Workflow

**Step 1** — Define questions:

```
Research "Acme Corp":
- Recent news and announcements
- Key executives
- Funding history
- Main competitors
```

**Step 2** — Gather information:

```
Use @Perplexity to search for recent news
Use @Exa to find relevant articles
```

**Step 3** — Synthesize:

```
Create a structured report:
- Executive summary
- Key findings per question
- Sources cited
```

## AI Search Tools

| Tool | Best for |
|------|----------|
| **Perplexity** | Current events, factual questions |
| **Exa** | Finding relevant documents and pages |
| **Firecrawl** | Extracting data from specific URLs |

### Perplexity

```
Use @Perplexity to answer: What is Acme Corp's latest product launch?
```

### Exa

```
Use @Exa to find the 10 most relevant articles about Acme Corp's AI strategy
```

### Web scraping

```
Scrape Acme Corp's team page to get all executive names and titles
```

## Example: Lead Research

**Step 1** — Input:

```
[user-input] Enter company name and website URL
```

**Step 2** — Research:

```
Use @Perplexity to find:
- Company size and industry
- Recent news
- Technologies they use
- Potential pain points
```

**Step 3** — Generate report:

```
Create a structured report:
- Company overview
- Key findings
- Recommended approach
- Sources
```

## Advanced Patterns

### Multi-source research

```
Search using @Perplexity
Also find articles via @Exa
Combine into a single report
```

### Structured extraction

```
Extract all mentions of:
- Company names (with context)
- Dollar amounts (with reference)
- Dates (with events)
```

### Human review

```
[user-input] Review findings:
- Approve and continue
- Request more research
- Edit the summary
```

## Tips

1. **Be specific** — Narrow questions get better results
2. **Use multiple sources** — Cross-reference information
3. **Verify facts** — AI can hallucinate, check sources
4. **Structure output** — Consistent formats for easy comparison
