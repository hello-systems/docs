## Connect

<Steps>
  <Step title="Go to Settings → Integrations → Firecrawl" />
  <Step title="Enter your Firecrawl API key" />
  <Step title="Done" />
</Steps>

## Actions

| Action | Example Prompt |
|--------|---------------|
| Scrape page | `Use @Firecrawl to scrape {{url}}` |
| Extract data | `Use @Firecrawl to extract product info from {{url}}` |
| Crawl site | `Use @Firecrawl to crawl all pages on {{domain}}` |

## Example

```
Use @Firecrawl to scrape {{url}} and extract:
- Title
- Main content
- All links
- Images

Output as JSON.
```
