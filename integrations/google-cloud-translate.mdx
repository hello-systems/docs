## Actions

| Action | Example Prompt |
|--------|----------------|
| Translate text | `Translate "{{text}}" to Spanish using @Google Cloud Translate` |
| Detect language | `Detect language of "{{text}}" using @Google Cloud Translate` |
| Batch translate | `Translate all items to French using @Google Cloud Translate` |
