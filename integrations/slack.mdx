|--------|---------------|
| Send message | `Send to @Slack #sales: "New lead: {{name}}"` |

| Rich message | `Send formatted message to @Slack with blocks` |

```

- Bold: *text*
- Code: `code`
- Link: <url|text>
- Emoji: :rocket:
```
