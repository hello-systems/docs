## Actions

| Action | Example Prompt |
|--------|----------------|
| Create presentation | `Create @Google Slides presentation titled "{{title}}"` |
| Add slide | `Add slide to @Google Slides {{presentationId}}` |
| Update text | `Update text in @Google Slides {{presentationId}}` |
