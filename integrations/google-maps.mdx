## Connect

<Steps>
  <Step title="Go to Settings → Integrations → Google Maps" />
  <Step title="Enter your Google Maps API key" />
  <Step title="Done" />
</Steps>

## Actions

| Action | Example Prompt |
|--------|----------------|
| Geocode address | `Get coordinates for "{{address}}" using @Google Maps` |
| Search places | `Search @Google Maps for "{{query}}" near {{location}}` |
| Get directions | `Get @Google Maps directions from {{origin}} to {{destination}}` |
| Get place details | `Get @Google Maps details for place {{placeId}}` |
