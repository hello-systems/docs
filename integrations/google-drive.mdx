## Actions

| Action | Example Prompt |
|--------|----------------|
| Upload file | `Upload {{file}} to @Google Drive folder "Reports"` |
| Download file | `Download @Google Drive file "data.csv"` |
| List files | `List files in @Google Drive folder "Projects"` |
| Share file | `Share @Google Drive file with {{email}}` |
