## Connect

<Steps>
  <Step title="Go to Settings → Integrations → Resend" />
  <Step title="Enter your Resend API key" />
  <Step title="Done" />
</Steps>

## Actions

| Action | Example Prompt |
|--------|----------------|
| Send email | `Send email via @Resend to {{email}} with subject "Welcome"` |
| Send template | `Send @Resend template "welcome" to {{email}}` |
| Batch send | `Send @Resend emails to all contacts in the list` |
