## Actions

| Action | Example Prompt |
|--------|----------------|
| Get events | `Get @Google Calendar events for next week` |
| Create event | `Create @Google Calendar event "Meeting" on {{date}} at {{time}}` |
| Update event | `Update @Google Calendar event {{eventId}} to new time` |
| Delete event | `Delete @Google Calendar event {{eventId}}` |
