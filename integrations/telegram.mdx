## Connect

<Steps>
  <Step title="Go to Settings → Integrations → Telegram" />
  <Step title="Enter your Telegram Bot token" />
  <Step title="Done" />
</Steps>

## Actions

| Action | Example Prompt |
|--------|----------------|
| Send message | `Send @Telegram message to chat {{chatId}}: "Alert!"` |
| Send photo | `Send @Telegram photo {{url}} to chat {{chatId}}` |
| Send document | `Send @Telegram file {{file}} to chat {{chatId}}` |
