## Actions

| Action | Example Prompt |
|--------|----------------|
| Upload file | `Upload {{file}} to @Dropbox folder "/reports"` |
| Download file | `Download @Dropbox file "/data/report.csv"` |
| List files | `List files in @Dropbox folder "/projects"` |
| Create folder | `Create @Dropbox folder "/archive/2024"` |
