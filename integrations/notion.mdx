## Connect

## Actions

| Action | Example Prompt |
|--------|---------------|
| Query database | `Get all items from @Notion database "Tasks"` |
| Create page | `Create page in @Notion with title "{{title}}"` |
| Update page | `Update @Notion page with new content` |
| Append block | `Append paragraph to @Notion page` |
