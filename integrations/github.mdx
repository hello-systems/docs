|--------|----------------|
| Create issue | `Create @GitHub issue in {{repo}}: "{{title}}"` |

| Get PRs | `Get @GitHub pull requests in {{repo}}` |
| Create PR | `Create @GitHub pull request in {{repo}} from {{branch}}` |
