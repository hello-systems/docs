## Actions

| Action | Example Prompt |
|--------|----------------|
| Add subscriber | `Add {{email}} to @Mailchimp list "Newsletter"` |
| Send campaign | `Send @Mailchimp campaign to list "Customers"` |
| Get stats | `Get @Mailchimp campaign stats for {{campaignId}}` |
| Update tags | `Add tag "VIP" to @Mailchimp subscriber {{email}}` |
