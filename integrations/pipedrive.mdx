## Actions

| Action | Example Prompt |
|--------|---------------|
| Get deals | `Get open deals from @Pipedrive` |
| Create deal | `Create deal in @Pipedrive for {{company}}` |
| Update stage | `Move @Pipedrive deal to next stage` |
| Add activity | `Add call activity to @Pipedrive deal` |
