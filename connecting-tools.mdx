## Add an Integration

<Steps>
  <Step title="Go to Settings → Integrations" />
  <Step title="Find your app → Click Connect" />
  <Step title="Authenticate (OAuth or API key)" />
  <Step title="Done — credentials stored encrypted" />
</Steps>

## Use in Workflows

Mention integrations with `@`:

```
Fetch contacts from @Google Sheets and add to @HubSpot
```

Script uses the right credentials and API calls automatically.

## Custom APIs

Connect to any REST API:

```
Fetch data from https://api.example.com/users with header Authorization: Bearer {{API_KEY}}
```

## Security

<Info>
- Credentials encrypted at rest
- OAuth tokens auto-refresh
- Revoke access anytime from Settings
